# 🚀 Launching CosmosView: My Next-Gen 3D Space & Satellite Tracker

I am incredibly excited to announce the launch of **CosmosView**, a passion project that merges my love for astronomy, orbital mechanics, and cutting-edge frontend engineering! 🌌🛰️

After weeks of development (and some intense pair-programming with AI), I set out with a clear goal: to build an immersive, highly performant, and interactive 3D universe right in the browser. 

CosmosView isn't just a static planetarium; it's a dynamic, real-time application featuring two core modules:

✨ **1. The Interactive Sky Observatory**
*   **Real-time celestial rendering** of stars, constellations, Deep Space Objects (DSOs), and the Milky Way.
*   Accurate **Astronomy calculations** representing the exact position of the Moon, Sun, and planets based on your geographical coordinates.
*   A responsive, glassmorphic UI that adapts flawlessly across Desktop and Mobile (including an adaptive split-screen layout for portrait devices).

🌍 **2. The Orbital Tracking System**
*   A fully interactive 3D holographic wireframe Earth.
*   **Live Satellite Tracking** utilizing the `satellite.js` library to propagate true TLE (Two-Line Element) data via the SGP4 algorithmic model. 
*   Visualizing thousands of objects in real-time—from the **ISS** to massive mega-constellations like **Starlink**—allowing you to see exact orbital trajectories and telemetry data dynamically wrapping around the globe.

🛠️ **The Tech Stack Behind the Magic:**
*   **React + Vite** for a blazing-fast SPA foundation.
*   **Three.js (@react-three/fiber & @react-three/drei)** to handle the complex 3D rendering pipeline and camera controls.
*   **Tailwind CSS** to build a modern, high-contrast "neon-cyberpunk" aesthetic.
*   **Celestrak APIs** for live orbital data and **Astronomy Engine** for high-precision celestial mechanics.

This project pushed me to master state management across 3D contexts, optimize rendering loops for 60fps performance on mobile, and implement complex mathematical models for orbital propagation.
